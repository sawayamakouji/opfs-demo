# OPFS Demo

Origin Private File System (OPFS) の機能をブラウザ上で体験するためのデモプロジェクトです。

**公開URL:** [https://sawayamakouji.github.io/opfs-demo/](https://sawayamakouji.github.io/opfs-demo/)

## 機能一覧

このデモでは、以下のOPFS関連の機能を試すことができます。

*   **基本的なファイル操作:**
    *   `test.txt` という名前のファイルを作成、書き込み、読み込み、削除します。

*   **大容量ファイルの操作:**
    *   指定したMBサイズの巨大なファイルを生成し、書き込みます。
    *   生成したファイルを読み込み、プレビューエリアに表示します。（※サイズが大きいとブラウザが固まる可能性があります）

*   **汎用的なファイル修正:**
    *   ファイル名を直接指定して、ファイルの新規作成、読み込み、保存（修正）、削除ができます。
    *   テキストエリアでファイル内容を自由に編集できます。

*   **大量CSVデータの生成と検索:**
    *   指定した行数（例: 20万行）の商品マスタCSVファイルを生成・書き込みします。
    *   生成した商品マスタの中から、特定の商品コードで商品を検索し、商品名を取得します。

## 使い方

1.  上記の公開URLにアクセスします。
2.  各機能セクションのボタンをクリックして、ファイル操作を試してください。
3.  操作の結果は、画面上のアラートやプレビューエリア、ブラウザの開発者ツール（F12）のコンソールで確認できます。

## 注意点

*   このデモで作成・保存される全てのファイルは、現在お使いのブラウザの専用領域（Origin Private File System）に保存されます。あなたのPCの他の場所には影響しません。
*   非常に大きなファイル（数百MB以上）を生成したり読み込んだりすると、ブラウザの動作が重くなったり、一時的に応答しなくなる場合があります。
